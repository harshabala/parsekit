## Methodology

- **Fixtures:** Synthetic, license-safe samples under `scripts/benchmark-fixtures/`.
- **Baseline:** Naive extraction — `pdftotext` when available, otherwise PDF text layer via `pypdf`; Office formats use raw OOXML text/cell dumps.
- **ParseKit:** `src-tauri/binaries/parsekit-sidecar-aarch64-apple-darwin` sidecar (`format: md`, `ocrEnabled: true`). Spreadsheets emit JSON per ParseKit rules.
- **Tokenizers:** `tiktoken` encodings `cl100k_base` and `o200k_base`.
- **Δ (delta):** Percent reduction from baseline to ParseKit output; floored at 0% when baseline is empty (scanned PDFs) or ParseKit output is larger.
